Manchester United 2024/2025 Premier League Dashboard | Power BI

Project Overview

    This project analyzes Manchester United’s 2024/25 Premier League season using match and player data sourced from FBref. The objective was to convert raw football statistics into interactive insights through Power BI.

Data Pipeline

    1. Data collected from FBref

    2. AI-assisted data cleaning and structuring

    3. Data modeling inside Power BI

    4. DAX measures created for KPIs and advanced metrics

Dashboard Structure

  1. Overview

    - Goals Scored: 44

    - Goals Conceded: 54

    - Points per Match: 1.11

    - Avg Possession: 53.5%

    - Passing Accuracy: 82.5%

  2. Offensive Analysis

    - Total xG: 52.6

    - Goals per 90: 1.11

    - Key contributors: Bruno Fernandes, Amad Diallo

    - Shot vs Shot-on-target comparisons

    - Goals vs xG correlation visualization

  3. Defensive Analysis

    - 493 Tackles Won

    - 386 Blocks

    - 400 Challenges

    - Defensive phase distribution analysis

    - Tackles vs Blocks correlation

  4. Passing & Progression

    - 406 Key Passes

    - Short Pass Completion: 90.2%

    - Medium Pass Completion: 88.2%

    - Long Pass Completion: 51.6%

    - Progressive passes comparison vs top teams

Key Findings

    - Underperformance relative to xG indicates finishing issues

    - Defensive action volume suggests reactive defensive approach

    - Creative output concentrated among limited players

    - Structural gap in progression compared to league leaders
